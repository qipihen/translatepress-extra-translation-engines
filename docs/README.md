# GitHub Pages Site (docs/)

This folder contains the project website for **TranslatePress Extra Translation Engines**.

## Files

- `index.html`: Landing page for product overview, features, installation, use cases, FAQ, and extension plan.
- `style.css`: Responsive styles for the static site.
- `en-US/` and `zh-CN/`: Existing technical documentation for users and developers.

## Deploy with GitHub Pages

1. Open your repository on GitHub.
2. Go to `Settings -> Pages`.
3. Under **Source**, select **Deploy from a branch**.
4. Choose branch: `main`.
5. Choose folder: `/docs`.
6. Click **Save**.
7. Wait for GitHub Pages build, then open the generated URL.

## Notes

- This is a static HTML/CSS site with no server-side dependencies.
- If you use a custom domain, update the canonical URL in `index.html`.
